# Java-Coding-Exercise--
Java Basic to Advanced

Boolean: Represents one of two logical values:true or false.It is commonly used in conditions and control statements.
    boolean booleanVar;

Byte Data Type: An 8-bit signed integer used to save memory in large numeric arrays.
    byte byteVar;

short Data Type: A 16-bit signed integer often used when memory is limited and values are moderate in size.
    short shortVar;

Int Data Type: A 32-bit signed integer and the most commonly used numeric data type.
    int intVar;  size: 4 bytes (32 bits)


Long Data Type: A 64-bit signed integer used when int is not sufficient for large values.
    long longVar;  size: 8 bytes (64 bits)
